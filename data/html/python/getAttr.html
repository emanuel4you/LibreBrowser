string getAttr(string key, string attr)

Determine the DCL value of this tile's attribute.

This function returns the value of the dialog box attribute. The key argument specifies the dialog box tile; key is a string and is case-sensitive.

The attribute argument is the name of the attribute. See the [link]DCL Programming Reference for valid attribute names.

This function returns the attribute as a string
