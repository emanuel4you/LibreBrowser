PYTHON Functions

bool newDialog(string name, int id)

Display a dialog box.

Use this function to display a dialog box.

The dialog box is identified by dialog within the DCL file named dcl-file. The string dialog is obtained from the loadDialog function.

This function returns True if successful otherwise, False.

NOTE

After you use newDialog the first time to initialize a dialog box, then use the newPositionedDialog function to display a dialog box at a specific screen location.
