is_tab_stop

DCL Attributes

is_tab_stop = true/false;

Specifies whether the tile receives keyboard focus when pressing the Tab key.

Possible values are true or false (default: true).

NOTE

If the tile is disabled, it is not a tab stop even if this attribute is true. If false, the tile is not a tab stop.

Valid Tiles

All active tiles

EXAMPLE

is_tab_stop = true;
