key

DCL Attributes

key = "string";

Specify the name of a tile, as used by a LISP or Python function.

This attribute specifies the name by which a LISP or [link]Python program to identify a tile. Each key must be distinct within the dialog box, and the string is case-sensitive.

string: a string.

EXAMPLE

key = "accept";
