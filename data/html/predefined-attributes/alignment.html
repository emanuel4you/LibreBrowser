alignment

DCL Attributes

alignment = position;

Specifies the horizontal or vertical positioning.

For a tile that is a descendant of a column, the acceptable values include left, right, or centered, with the default setting being left.

For a tile that is a descendant of a row, the permissible values are top, bottom, or centered, with the default setting being centered.

EXAMPLE

alignment = top;

Valid Tiles

All tiles
