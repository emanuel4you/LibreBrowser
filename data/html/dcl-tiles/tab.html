DCL Tiles

: tab

Display a widgets in tab.

DCL Active Tile

This tile fits to the register tile

NOTE

The tab tile is specifically designed for the DCL implementation included with LibreDcl+.

Valid Attributes

: tab {

alignment = position;

children_alignment = position;

children_fixed_height = flag;

children_fixed_width = flag;

fixed_height = flag;

fixed_width = flag;

height = number;

label = "string";

width = number;

}

EXAMPLE

: tab {

: button {

action = "(zoomcmd)";

key = "BZ";

label = "Zoom";

}

: button {

action = "(pancmd)";

key = "BP";

label = "Pan";

}

}
