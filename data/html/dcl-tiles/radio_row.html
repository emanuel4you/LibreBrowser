radio_row

DCL Tiles

: radio_row

Display a row of radio buttons. DCL Cluster Tile

This cluster tile organizes one or more radio buttons in a horizontal arrangement. A radio button tile must be positioned either within this tile or within a radio column cluster tile. If not, the program will be unable to determine the relationships among the radio buttons.

NOTE

A radio row cluster is exclusively for radio buttons. For other types of tiles, please utilize the row cluster.

Valid Attributes

: radio_row {

alignment = position;

children_alignment = position;

children_fixed_height = flag;

children_fixed_width = flag;

fixed_height = flag;

fixed_width = flag;

height = number;

label = "string";

width = number;

}

EXAMPLE

: radio_row {

label = "3D Viewpoints";

: radio_button {

action = "(select-me)";

key = "R1";

label = "Value 1";

}

: radio_button {

action = "(select-me)";

key = "R2";

label = "Value 2";

}

}

NOTE

The value attribute signifies which radio button is currently selected, represented by the value of "1":

value = "1"
