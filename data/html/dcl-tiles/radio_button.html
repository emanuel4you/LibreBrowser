radio_button

DCL Tiles

: radio_button

Display a radio button.

DCL Active Tile

This active tile features a radio button that indicates selection through the appearance of a dot when activated. The unique characteristic of the radio button allows for the selection of only one option at a time; selecting one radio button will deactivate any other radio button within the same group.

It is essential to position a radio_button tile within a radio_column or radio_row cluster tile. Failure to do so will result in the program being unable to determine the relationships among the radio buttons.

To enable users to select multiple options, it is advisable to utilize check boxes (through the toggle tile) instead.

NOTE

When creating a DCL file that includes multiple radio buttons within a radio_column or radio_row cluster that are activated (value = "1"), the program will disregard all subsequent "turned-on" radio buttons after the first.

Valid Attributes

: radio_button {

action = "(string)";

alignment = position;

fixed_height = flag;

fixed_width = flag;

height = number;

is_enabled = flag;

is_tab_stop = flag;

key = "string";

label = "string";

mnemonic = "char";

value = "string";

width = number;

}

EXAMPLE

: radio_button {

action = "(option_1)";

key = "R1";

label = "Option 1";

}

NOTE

The value attribute signifies which radio button is currently selected, indicated by the value of "1":

value = "1".
