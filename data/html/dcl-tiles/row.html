row

DCL Tiles

: row

Display a row of horizontal tiles.

DCL Cluster Tile

This cluster tile arranges a sequence of tiles in a horizontal alignment without any borders. The row cluster is capable of accommodating any tile type, with the exception of a standalone radio button; for grouping radio buttons, it is advisable to utilize either the radio_column or radio_row cluster tiles.

NOTE

To enclose the column of tiles within a box, use the boxed_row cluster.

Valid Attributes

: row {

alignment = position;

children_alignment = position;

children_fixed_height = flag;

children_fixed_width = flag;

fixed_height = flag;

fixed_width = flag;

height = number;

label = "string";

width = number;

}

EXAMPLE

: row {

: button {

action = "(zoomcmd)";

key = "BZ";

label = "Zoom";

}

: button {

action = "(pancmd)";

key = "BP";

label = "Pan";

}

}
