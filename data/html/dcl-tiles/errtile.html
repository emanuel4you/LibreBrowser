Display an error notification at the lower section of a dialog box.

DCL Error Handler Tile

This error handler tile presents a concise error notification at the bottom of a dialog box. To display a message within the errtile, assign a value to the tile associated with the key "error." By default, the errtile is empty.

It is important to note that the format of this tile is distinct from that of most other tiles, as it does not include a colon (:) prefix or a brace ({}) suffix.

Valid Attributes

none

EXAMPLE

errtile;
