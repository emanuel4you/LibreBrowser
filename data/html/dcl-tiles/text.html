DCL Tiles

: text

Display a string of text. DCL Text Tile

This text tile presents a sequence of text. It is recommended to utilize the text tile in conjunction with the concatenation and paragraph clusters to form horizontal and vertical arrangements of text, respectively.

NOTE

It is important to note that most tiles possess their own title (through the label attribute), making this function infrequently necessary.

Valid Attributes

: text {

alignment = position;

fixed_height = flag;

fixed_width = flag;

height = number;

is_bold = flag;

key = "string";

label = "string";

value = "string";

width = number;

}

EXAMPLE

: text {

key = "T12";

label = "How to use progeCAD:";

}
