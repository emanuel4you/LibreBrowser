DCL Tiles

: text_part

Display multiple segments of text. DCL Text Tile

This text tile is utilized in conjunction with the concatenation and paragraph tiles to present various segments of text.

Valid Attribute

: text_part {

label = "string";

}

EXAMPLE

: concatenation {

text_part {label = "This";}

text_part {label = " is";}

text_part {label = " text.";}

}
