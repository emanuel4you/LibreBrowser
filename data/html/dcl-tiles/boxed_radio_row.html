boxed_radio_row

DCL Tiles

: boxed_radio_row

A rectangle can be drawn around a series of radio buttons, with the option to include a title.

DCL Cluster Tile

This cluster tile creates a border surrounding a series of radio buttons. If the label attribute is absent or a null string ("") is provided, the border will be displayed without a title.

NOTE

  • For a series of radio buttons that do not require a border or title, utilize the radio_row cluster.

  • To encase a row of other types of tiles (not radio buttons) within a border, employ the boxed_row cluster.

  • The [link]value attribute specifies which radio button is selected by default, indicated by the value of "1":

value = "1".

Valid Attributes

: boxed_radio_row {

alignment = position;

children_alignment = position;

children_fixed_height = flag;

children_fixed_width = flag;

fixed_height = flag;

fixed_width = flag;

height = number;

label = "string";

value = "string";

width = number;

}

EXAMPLE

: boxed_radio_row {

label="Edit Corner"

: radio_button {

action = "(chamferlsp)";

key = "R5";

label = "Chamfer";

}

: radio_button {

action = "(filletlsp)";

key = "R6";

label = "Fillet";

}

}
