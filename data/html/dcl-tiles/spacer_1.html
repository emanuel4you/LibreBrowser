Insert a unit blank tile with dimensions of 1 in both width and height.

DCL Decorator Tile

This decorator tile introduces a blank tile with the attributes of height and width configured to 1.

NOTE

It is important to observe that the structure of this tile is distinct from that of most other tiles: it lacks a : (colon) prefix and a { ... } (brace) suffix.

Valid Attributes

none

EXAMPLE

spacer_1;
