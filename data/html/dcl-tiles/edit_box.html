edit_box

DCL Tiles

: edit_box

The user is prompted to enter a line of text.

DCL Active Tile

This active tile features an edit box that invites the user to input either a number or text. The text within the edit box is aligned to the left. The program appends a null character (\0) at the end of the string entered by the user. If an edit_limit is set and the maximum character count is reached, all user inputs, except for backspace and delete, will be disregarded.

Valid Attributes

: edit_box {

action = "(string)";

alignment = position;

allow_accept = flag;

edit_limit = int;

edit_width = number;

fixed_height = flag;

fixed_width = flag;

height = number;

is_enabled = flag;

is_tab_stop = flag;

key = "string";

label = "string";

lower_only = flag;

mnemonic = "char";

password_char = "char";

value = "string";

width = number;

}

EXAMPLE

: edit_box {

action = "(texted)";

allow_accept = true;

edit_limit = 31;

key = "EB";

label = "File name: ";

mnemonic = "F";

value = "drawing";

}

NOTE

  • The label attribute specifies the title of the edit box, which is aligned to the left.

  • The value attribute contains the initial text shown in the edit box, also left-justified.
