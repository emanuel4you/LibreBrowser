column

DCL Tiles

: column

Position a column of tiles, which may include an optional title

DCL Cluster Tile

This cluster tile organizes a sequence of tiles in a vertical alignment without any borders. The column cluster is capable of accommodating any tile type, with the exception of a standalone radio_button. To enclose the column of tiles within a border, utilize the boxed_column cluster.

NOTE

Columns will not feature titles unless they are enclosed in a box.

Valid Attributes

: column {

alignment = position;

children_alignment = position;

children_fixed_height = flag;

children_fixed_width = flag;

fixed_height = flag;

fixed_width = flag;

height = number;

label = "string";

width = number;

}

EXAMPLE

: column {

: button {

action = "(linecmd)";

key = "BL";

label = "Line";

}

: button {

action = "(arccmd)";

key = "BA";

label = "Arc";

}

}
