toggle

DCL Tiles

: toggle

Display a check box.

DCL Active Tile

This interactive tile features a check box. When activated, the check box displays a check mark, and the value attribute returns 1. The text corresponding to the label attribute is positioned to the right of the box.

Valid Attributes

: toggle {

action = "(string)";

alignment = position;

fixed_height = flag;

fixed_width = flag;

height = number;

is_enabled = flag;

is_tab_stop = flag;

key = "string";

label = "string";

mnemonic = "char";

value = "string";

width = "string";

}

EXAMPLE

: toggle {

action = "(osnap)";

label = "Object Snap Mode";

value = "1"

}

NOTE

The value attribute indicates whether the check box is currently selected (indicated by a check mark), via the value of "1":

value = "1"
