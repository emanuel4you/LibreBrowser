DCL Tiles

: spacer

Insert a blank tile.

DCL Decorator Tile

This decorative tile creates an empty space within a dialog box. Although a spacer can be utilized to assist in the arrangement of other tiles, this particular tile is generally unnecessary, as the program typically arranges all tiles automatically with suitable spacing.

Valid Attributes

: spacer {

alignment = position;

fixed_height = flag;

fixed_width = flag;

height = number;

width = number;

}

EXAMPLE

: spacer {

height = "5";

width = "10";

}
