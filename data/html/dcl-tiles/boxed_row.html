boxed_row

DCL Tiles

: boxed_row

Enclose a series of tiles within a rectangle, optionally including a title.

DCL Cluster Tile

This cluster tile creates a border surrounding a series of tiles, excluding radio buttons. The border will be displayed without a title if the label attribute is absent or if an empty string ("") is provided.

NOTE

  • For a series of tiles that do not require a border or title, utilize the row cluster.

  • To enclose a series of radio buttons within a border, employ the boxed_radio_row cluster.

Valid Attributes

: boxed_row {

alignment = position;

children_alignment = flag;

children_fixed_height = flag;

children_fixed_width = flag;

fixed_height = flag;

fixed_width = flag;

height = number;

label = "string";

width = number;

}

EXAMPLE

: boxed_row {

label="Select Objects";

: button {

action = "(linelsp)";

key = "BB";

label = "Line";

}

: button {

action = "(circlelsp)";

key = "BW";

label = "Circle";

}

}
