DCL Tiles

: register

Display a register of tabs.

DCL Cluster Tile

This cluster tile arranges a sequence of tab tiles in a horizontal order. The register cluster only the tab tile.

NOTE

The register tile is specifically designed for the DCL implementation included with LibreDcl+.

Valid Attributes

: register {

label = "string";

}

EXAMPLE

: register {

: tab {

label = "Tab 1";

}

: tab {

label = "Tab 2";

}

}
