(empty? coll)

Returns true if coll has no items.

This function Returns true if coll has no items - same as (not (seq coll)).

NOTE

Please use the idiom (seq x) rather than (not (empty? x))

EXAMPLES

Code Returns
(empty? '()) true
(empty? '(1)) false
