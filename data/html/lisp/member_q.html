(member? coll x)

Returns true if x is member off coll true, false otherwise.

This function evaluates whether x is member off coll, if it is not in coll, the function will return false.

EXAMPLES

Code Returns
(member? '("one" "two" "three") "one") true
(member? '("one" "two" "three") "six") false
