mode_tile

LISP Functions

(mode_tile key mode)

Adjust the mode of the dialog box's title.

The argument accepts one of the following integer values:

Mode Meaning

0 Enable tile.

1 Disable tile.

2 Set the focus to the tile.

3 Select the contents of an edit box.

EXAMPLE

(mode_tile pt1 2)
