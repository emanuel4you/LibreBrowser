(reset! atom f)

Atomically resets the value of atom

Sets the value of atom to newval without regard for the current value. Returns newval.

EXAMPLE

Code Returns
(def! x (atom 10)) (atom 10)
(reset! x 20) 20
!x (atom 20)) !x (atom 20)
