command

LISP Functions

(command cmd [arguments] ...)

Exec this LibreCAD command.

This function allows the utilization of any LibreCAD command within a LISP program. The subsequent arguments will be employed for this purpose. It is essential to assign points to variables following the invocation of the function (get)point.

Certain commands may extend beyond the specified points. To terminate a command, input a space enclosed in quotes to signify an Enter, thereby concluding the command.

EXAMPLES

Code Returns
(command "arc" p1 p2 p3) Using preset variables.
(command "line" p1 p2 p3 " ") Space to end the command.
