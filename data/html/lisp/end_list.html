LISP Functions

(end_list)

End the handling of a dialog box list.

This function indicates the completion of processing a list in the current context.

EXAMPLE

Code Returns
(end_list) nil

NOTE

This function is also applicable when developing list boxes and drop-down list boxes in [link]DCL. For examples demonstrating the use of this function alongside DCL, refer to (list)_box and [link]popup_list (drop-down list box).
