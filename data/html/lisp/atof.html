LISP Functions

(atof string)

Convert the string to a real number.

Returns 0.0 if the string's content cannot be identified as a numerical value.

EXAMPLES

Code Returns
(atof "59") 59.0
(atof "213.35") 213.350
(atof ".1") 0.100000
(atof "Hello") 0.000000
