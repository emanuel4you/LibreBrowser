LISP Functions

(numberp item)

Determine whether an item is a number.

Assess whether the given item qualifies as a number. If the item is identified as a number, a T is returned; otherwise, nil is returned.

EXAMPLES

Code Returns
(numberp 0.2653) T
(numberp '(5 3.2 6)) nil
(numberp "Name") nil
