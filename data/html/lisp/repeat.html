repeat

LISP Functions

(repeat number statement1 [statement2 ...])

Repeat it x times.

This function reiterates the statements a specified number of times, as indicated by the number.

EXAMPLE

Code Returns
(setq value 0) 0
(repeat 5 (1+ value)(setq value (1+ value))) 5
