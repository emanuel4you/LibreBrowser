LISP Functions

(ver)

Returns the version number of LibreCAD's LISP.

This function returns the version number of the LISP programming

language included in LibreCAD.

EXAMPLE

Code Returns
(ver) LibreLisp 1.1.10 (devel, Jan 4 2025, 01:17:41) [GCC Ubuntu Clang 18.1.3 (1ubuntu1)] on linux
